Kebijakan Privasi Rak Buku Tracker

Terakhir diperbarui: 10 Juli 2026

Rak Buku Tracker ("Aplikasi") adalah aplikasi pencatat koleksi dan progres membaca buku pribadi. Kebijakan Privasi ini menjelaskan data apa saja yang dikumpulkan, bagaimana data tersebut digunakan, dan hak kamu sebagai pengguna.

1. Data yang Kami Kumpulkan

Untuk menjalankan fitur Aplikasi, kami mengumpulkan data berikut:

  • Data akun: nama, alamat email, dan kata sandi (disimpan dalam bentuk terenkripsi/hash, tidak pernah disimpan sebagai teks biasa).
  • Data buku: judul, penulis, ISBN, jumlah halaman, kategori, status baca, progres halaman, rating, dan catatan/review yang kamu tulis sendiri.
  • Data statistik membaca: target membaca tahunan dan riwayat aktivitas membaca di dalam Aplikasi.
  • Data perangkat terbatas: akses kamera digunakan khusus untuk memindai barcode ISBN buku, dan tidak menyimpan atau mengunggah foto/video apa pun.

2. File PDF yang Kamu Unggah

Aplikasi memungkinkan kamu mengunggah file PDF pribadi untuk dibaca di dalam Aplikasi. File ini disimpan secara lokal di perangkat kamu saja dan tidak diunggah atau dikirim ke server kami. Kami tidak memiliki akses ke isi file PDF yang kamu simpan.

3. Layanan Pihak Ketiga yang Digunakan

Aplikasi terhubung ke layanan pihak ketiga berikut untuk fitur pencarian dan pembacaan buku:

  • Google Books API — digunakan untuk mencari data buku dan melakukan pencarian berdasarkan ISBN. Silakan lihat Kebijakan Privasi Google.
  • Project Gutenberg (Gutendex API) — digunakan untuk mengambil teks buku domain publik yang dapat dibaca langsung di dalam Aplikasi.

Kami tidak mengontrol praktik privasi dari layanan pihak ketiga tersebut.

4. Bagaimana Kami Menggunakan Data Kamu

  • Menyediakan dan menjalankan fitur inti Aplikasi (pelacakan koleksi buku, review, statistik membaca).
  • Autentikasi dan menjaga keamanan akun kamu.
  • Mengirimkan notifikasi pengingat membaca secara lokal di perangkat kamu (notifikasi ini dijadwalkan langsung di HP kamu, bukan dikirim dari server kami).

Kami tidak menjual data pribadi kamu kepada pihak mana pun, dan tidak menampilkan iklan di dalam Aplikasi.

5. Penyimpanan Data

Data akun dan data buku kamu disimpan di server basis data milik pengembang Aplikasi. Kami menerapkan langkah keamanan yang wajar untuk melindungi data tersebut, namun tidak ada metode transmisi atau penyimpanan elektronik yang 100% aman.

6. Hak Kamu

Kamu dapat menghapus buku, review, atau data lain kapan saja langsung melalui Aplikasi. Untuk permintaan penghapusan akun secara menyeluruh atau pertanyaan lain terkait data pribadi kamu, silakan hubungi kami melalui kontak di bawah.

7. Privasi Anak

Aplikasi ini tidak ditujukan secara khusus untuk anak-anak di bawah 13 tahun, dan kami tidak dengan sengaja mengumpulkan data pribadi dari anak-anak di bawah usia tersebut.

8. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan tercermin melalui tanggal "Terakhir diperbarui" di atas.
